# Sentiment Analysis

This project aims to ingest tweets from Twitter's Stream API filtered by "overwatch", apply sentiment analysis on the tweet, and determine real-time sentiment for Blizzard's Overwatch.

# State of Project

Currently streaming in tweets and outputting:
  * Sentiment
  * Subjectivity of Sentiment
  * GMT Time and Date of tweet

 Future work includes:
  * Stream live data and graph to webpage
  * Stream through Spark to transform RDDs to view historical sentiments as well as real-time
